# WinSight: IPL Winning Factors Analytics

## Key Insights

| Factor | Impact |
|----------|----------|
| Toss Winner Win % | 51.47% |
| Score 200+ Win % | 85.87% |
| Avg Death Overs Runs (Winners) | 54.53 |
| Avg Death Overs Runs (Losers) | 45.30 |

## Overview

This project analyzes 225,000+ IPL ball-by-ball records and 950 IPL matches to identify factors influencing match outcomes.

## Key Findings

### Toss Impact
- Toss winners won only 51.47% of matches.

### First Innings Score
- Teams scoring 200+ successfully defended their total in 85.87% of matches.

### Death Overs
- Winning teams scored ~9 more runs in death overs than losing teams.

## Dataset

- IPL Matches (2008–2022)
- IPL Ball-by-Ball Data (2008–2022)
